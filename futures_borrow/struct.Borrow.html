pub struct Borrow<T> { /* fields omitted */ }

A mutable memory location with future-aware dynamically checked borrow rules.

Safe borrowing of data across Future tasks requires that the data is stored in stable memory. To do this, Borrow internally creates an Arc to store the data.

See crate level documentation for more details.

Returns Ready when the value is not already borrowed.

When Ready is returned, the next call to poll_borrow or try_borrow is guaranteed to succeed. When NotReady is returned, the current task will be notified once the outstanding borrow is released.
